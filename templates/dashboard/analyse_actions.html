Verdict synthétique
Relative Strength Index : oscillateur borné [0–100] mesurant la vitesse du momentum. Seuils classiques : > 70 suracheté · < 30 survendu.
Stochastique : compare la clôture à l'amplitude du range sur N périodes. Stochastique RSI applique la même formule au RSI. Seuils : > 80 · < 20.
CCI : écart à la moyenne mobile rapporté à la déviation absolue. Zones extrêmes ±100 / ±200.
Williams %R : position de la clôture dans le range [−100 ; 0]. > −20 suracheté · < −80 survendu.
Momentum / ROC / TRIX : mesurent la vitesse de variation du prix. Au-dessus de 0 = momentum haussier ; sous 0 = momentum baissier. TRIX filtre le bruit (triple EMA).
Moyennes mobiles : filtrent le bruit pour révéler la tendance. Golden cross = courte croise au-dessus de longue (haussier) ; Death cross = inverse. EMA réagit plus vite que SMA. WMA pondère linéairement, DEMA/TEMA réduisent le retard.
MACD : différence de deux EMA. La ligne de signal est l'EMA(9) du MACD. Croisement MACD > signal = signal haussier, inverse = baissier. PPO = MACD exprimé en pourcentage (comparable entre actions).
ADX : mesure la force de la tendance (peu importe la direction). > 25 tendance forte · < 20 absence de tendance. +DI > −DI = pression haussière.
Parabolic SAR : stops dynamiques qui suivent la tendance — points sous le prix = haussier, au-dessus = baissier.
Ichimoku Kinko Hyo : nuage (Senkou A/B), Tenkan-sen (court terme), Kijun-sen (moyen terme), Chikou (rétro-projection). Prix au-dessus du nuage = tendance haussière.
Bollinger Bands : SMA(N) ± k × écart-type. %B > 1 = cassure haute (suracheté) · %B < 0 = cassure basse (survendu). Compression du bandwidth = forte cassure imminente.
ATR : Average True Range — fourchette moyenne quotidienne (volatilité absolue). Stop-loss courant suggéré ≈ cours − 2 × ATR. ATR % normalise par le prix et permet la comparaison entre actions.
OBV : volume cumulé pondéré par le sens du prix — confirme/dénie la tendance.
MFI : RSI pondéré par le volume. > 80 suracheté · < 20 survendu.
A/D (Accumulation / Distribution) : pression acheteuse cumulée selon la position de la clôture dans le range.
